# GeoPlan 页面设计规范文档

## 一、项目概述

### 1.1 项目定位

**GeoPlan** 是一个面向新能源充电桩领域的**全网博弈与商业选址决策支持平台**，旨在帮助政府规划部门、投资运营商和车主用户进行充电基础设施的科学布局与决策分析。

### 1.2 目标用户

| 用户角色 | 核心需求 | 权限范围 |
|---------|---------|---------|
| **车主 (Owner)** | 查询充电站位置、查看覆盖范围、反馈需求 | 基础查询、反馈提交 |
| **投资商 (Investor)** | 选址分析、覆盖评估、方案对比 | 沙盒模拟、数据分析 |
| **管理员 (Admin)** | 系统管理、用户管理、数据维护 | 全部功能权限 |

### 1.3 技术架构

前端技术栈：
├── Vue 3.4+ (Composition API)
├── Vue Router 4.x
├── Pinia 2.x (状态管理)
├── Element Plus 2.x (UI组件库)
├── OpenLayers 9.x (地图渲染)
├── ECharts 5.x (数据可视化)
└── Vite 5.x (构建工具)

## 二、设计风格指南

### 2.1 设计理念

**关键词**：科技感、专业性、易用性、数据驱动

- **科技感**：通过渐变色彩、毛玻璃效果、流畅动效传递数字化平台属性
- **专业性**：清晰的数据可视化、严谨的布局结构、专业的图表呈现
- **易用性**：直观的交互设计、清晰的信息层级、完善的引导流程
- **数据驱动**：以地图和图表为核心载体，让数据说话

## 五、页面详细设计

### 5.1 登录页 (Login Page)

**页面结构**：
- 全屏渐变背景（深色主题 + 径向光晕）
- 居中登录卡片（宽度 420px，圆角 16px）
- Logo + 品牌名 + 标语
- 用户名/密码输入框
- 登录按钮
- 演示账号提示

**设计规格**：
- Logo 尺寸：40px
- 品牌名字号：28px（渐变文字）
- 标语字号：13px
- 输入框尺寸：Large
- 登录按钮：100% 宽度

### 5.2 地图展示与查询页 (Map View)

**页面结构**：
- 左侧控制面板（宽度 280px）：搜索框、图层控制、充电模式、区域统计
- 右侧地图区域：OpenLayers 地图展示充电站、小区、反馈点
- 充电站详情弹窗

**交互设计**：
- 搜索：支持回车搜索和点击搜索
- 图层切换：实时更新地图显示
- 模式切换：切换快充/慢充模式
- 点击充电站：弹出详情弹窗

### 5.3 充电覆盖分析页 (Coverage View)

**页面结构**：
- 左侧控制面板（宽度 280px）：分析参数、执行按钮、分析结果统计
- 右侧上方：地图区域（覆盖多边形 + 盲区标记）
- 右侧下方：图表区域（柱状图 + 仪表盘）

**图表设计**：
- 区域覆盖率柱状图：颜色 #00C896
- 充电站饱和度仪表盘：环形仪表盘，颜色 #00C896

### 5.4 商业选址沙盒页 (Sandbox View)

**页面结构**：
- 左侧控制面板（宽度 320px）：沙盒说明、站点配置表单、实时分析结果
- 右侧上方：地图区域（真实站点 + 虚拟站点标记）
- 右侧下方：图表区域（对比柱状图 + 饼图）

**交互设计**：
- 点击地图：添加虚拟站点
- 拖拽虚拟站点：实时更新位置
- 300ms 防抖重新计算覆盖

### 5.5 待开发页面设计方案

#### 5.5.1 多方案博弈对比页 (Sandbox Compare)

**设计概念**：双屏分屏 + 镜像绑定 + 雷达图评分

**核心交互**：
- 双屏镜像绑定：缩放、平移同步
- 方案保存/加载：支持多方案管理
- 雷达图实时对比：自动计算各维度评分

#### 5.5.2 公众参与页 (Feedback View)

**设计概念**：需求打点 + 评价 + 热力图

**核心功能**：
- 地图点击打点提交需求
- 反馈热力图可视化
- 反馈列表与详情
- 星级评价系统

#### 5.5.3 AI 辅助决策页 (AI Assistant)

**设计概念**：智能对话 + 方案报告 + Text-to-GIS 语义检索

**核心功能**：
- 自然语言对话：支持选址咨询、数据分析提问
- 方案报告生成：自动生成 PDF 格式选址报告
- Text-to-GIS：语义化查询地理信息

## 七、响应式设计规范

### 7.1 断点设置

| 断点 | 宽度 | 布局策略 |
|-----|------|---------|
| 桌面端 | ≥ 1200px | 完整布局，侧边栏展开 |
| 平板端 | 768px - 1199px | 侧边栏收起，内容自适应 |
| 移动端 | < 768px | 侧边栏抽屉式，内容堆叠 |

### 7.2 适配策略

- 侧边栏：小屏幕自动收起
- 控制面板：小屏幕转为折叠面板
- 地图：始终占据全屏宽度
- 图表：小屏幕转为纵向堆叠
